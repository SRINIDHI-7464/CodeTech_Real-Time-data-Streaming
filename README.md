1. Introduction

Real-Time Data Streaming is an important application of Data Analytics and Internet of Things (IoT) that enables continuous collection, processing, and visualization of data as it is generated. Real-time streaming is widely used in smart cities, healthcare, industrial automation, environmental monitoring, transportation, and financial systems for instant decision-making.

In this project, Python is used to simulate real-time streaming of environmental sensor data obtained from the Environmental Sensor Telemetry dataset. The dataset contains sensor readings such as temperature, humidity, carbon monoxide (CO), smoke, light intensity, LPG concentration, and motion detection. Data preprocessing, exploratory data analysis (EDA), visualization, and live streaming simulation were performed to monitor sensor values continuously. This project demonstrates how real-time analytics can support monitoring systems and enable faster responses to changing environmental conditions.

3. Dataset Description

The dataset used for this project was downloaded from Kaggle and contains environmental sensor telemetry data collected from IoT devices.

The dataset includes the following attributes:

* Timestamp (ts)
* Device ID
* Temperature
* Humidity
* Carbon Monoxide (CO)
* Smoke
* LPG
* Light
* Motion

The dataset is stored in CSV format and is widely used for IoT analytics, sensor monitoring, and real-time streaming applications.
